Persistamp

A productivity/habit-tracking web application built to promote a healthy lifestyle with positive reinforcement! Create and track your habits, analyze your productivity over time with graphs and statistics, earn points by keeping up with your habits, stay motivated by utilizing your custom reward shop and redeem those hard-earned points!

CardMe

A flashcard web application that allows users to create and maintain customized flashcards or navigate sets created by fellow users! Users can 'favorite' sets to bookmark them for later or 'like' sets to show support. Features include a randomized, multiple-choice quiz and a search that filters particular subjects, sets, cards, and users!

Technologies: JavaScript, React, Redux, CSS3, Docker, Python, Flask, SQLAlchemy ORM, PostgreSQL

GitHub Live Site

Mediumm

A blogging web application, clone of the popular site Medium, that allows users to read and write 'stories' from our network of bloggers! Users can comment on stories, support stories or comments by giving them a 'clap', and follow specific content creators to be alerted when new stories are released!

Technologies: JavaScript, Pugjs, CSS3, Express.js, Node.js, Sequelize ORM, PostgreSQL

An expense-tracking web application that allows users to record their personal expenses and financial data! Users can create new expenses, settle owed expenses, and request payment from fellow users. Users have the ability to 'friend' other users, cash out their current balance, and notify users of unpaid expenses!

Technologies: JavaScript, React, Redux, CSS3, Express.js, Node.js Sequelize ORM, PostgreSQL
